 ######### old version
Use these in **Windows Command Prompt (`cmd.exe`)** from repo root:

1. Create and activate venv
```cmd
python -m venv .venv
.venv\Scripts\activate.bat
pip install -e .[dev]
```

2. Prepare synthetic data + run DQ pipeline
```cmd
set PYTHONPATH=src
python -m pipeline.run_demo --start-date 2019-01-01 --end-date 2024-12-31 --num-factors 400 --universe-name CP_RATE_CORP
```

3. Start Dash UI
```cmd
set PYTHONPATH=src
python -m ui.app --results-path data/processed/dq_results --raw-path data/raw/timeseries_raw --membership-path data/processed/universe_membership
```

4. Open browser
```text
http://127.0.0.1:8050
```

If you open a new `cmd` window, reactivate venv and re-run `set PYTHONPATH=src`.
